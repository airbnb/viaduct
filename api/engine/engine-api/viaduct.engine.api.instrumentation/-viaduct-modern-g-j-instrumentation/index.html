ViaductModernGJInstrumentation

interface ViaductModernGJInstrumentation : Instrumentation

ViaductModernGJInstrumentation is an implementation of GraphQL Java's Instrumentation interface that contains the additional methods required by ViaductModernInstrumentation.

This allows us to adapt GraphQL Java's Instrumentation to an instrumentation that can be used inside Viaduct's execution strategy.
